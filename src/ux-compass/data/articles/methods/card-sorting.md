# Карточная сортировка

**Карточная сортировка** (Card Sorting) — это качественный метод исследования, при котором пользователи группируют и назвают темы по своему пониманию структуры информации.

## Когда применяется

- **Валидация**: проверка навигации до разработки
- **Информационная архитектура**: проектирование структуры сайта/приложения
- **Оптимизация**: улучшение навигации существующего продукта

## Цели исследования

✅ **Понимание ментальных моделей** — как пользователи думают о категоризации  
✅ **Валидация структуры** — проверка текущей навигации  
✅ **Генерация таксономии** — создание системы категорий  
✅ **Улучшение UX** — оптимизация поиска и навигации

## Преимущества

✅ **Пользовательская перспектива** — структура с точки зрения пользователя  
✅ **Обнаружение инсайтов** — выявление неожиданных связей между темами  
✅ **Валидация гипотез** — проверка предположений о категоризации  
✅ **Улучшение навигации** — конкретные рекомендации по структуре

## Недостатки

❌ **Сложность модерации** — требуется опыт проведения сессий  
❌ **Время на анализ** — нужно время для кодирования данных  
❌ **Ограниченная масштабируемость** — обычно 5-8 пользователей  
❌ **Субъективность** — интерпретация зависит от исследователя

## Процесс проведения

1. **Подготовка карточек** — создание тем для сортировки
2. **Разработка сценариев** — определение задач для пользователей
3. **Рекрутинг участников** — подбор пользователей по критериям
4. **Проведение сессий** — пользователи сортируют карточки
5. **Кодирование данных** — категоризация группировок
6. **Анализ результатов** — выявление паттернов
7. **Формулировка рекомендаций** — структура навигации

## Методология

- **Open card sorting** — пользователи сами создают структуру
- **Closed card sorting** — пользователи сортируют по готовой структуре
- **Online vs in-person** — дистанционные или очные сессии

## Типичные метрики

| Метрика | Описание |
|---------|----------|
| Success Rate | Процент успешно выполненных задач |
| Time on Task | Время на сортировку карточек |
| Consistency Score | Степень согласованности группировок |
| Dendrogram Analysis | Визуализация иерархии категорий |

## Стоимость и сроки

- **Минимум**: 5 дней (для быстрого теста с панелью)
- **Максимум**: 21 день (для глубокого исследования)
- **Доступ**: own/panel (собственная панель или внешняя)
- **Бюджет**: low/mid (зависит от размера панели)

## Сложность

**Medium** — требует навыков модерации и анализа качественных данных.
