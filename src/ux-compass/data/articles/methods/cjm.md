# CJM (Customer Journey Map)

**CJM (Customer Journey Map)** — это метод исследования, при котором создаётся детальная карта пути клиента через продукт/услугу с точки зрения пользователя.

## Когда применяется

- **Валидация**: понимание полного опыта клиента
- **Оптимизация**: выявление точек трения в пользовательском пути
- **Стратегия**: разработка кросс-канальных решений

## Цели исследования

✅ **Понимание опыта** — полный взгляд на путь клиента  
✅ **Выявление барьеров** — обнаружение проблемных зон  
✅ **Синхронизация команд** — общее понимание пользовательского пути  
✅ **Приоритизация улучшений** — фокус на самых важных проблемах

## Преимущества

✅ **Комплексность** — видит полную картину опыта  
✅ **Визуализация** — понятное представление для всех стейкхолдеров  
✅ **Междисциплинарность** — объединяет маркетинг, продукт, дизайн  
✅ **Фокус на пользователе** — смещение акцента с функций на опыт

## Недостатки

❌ **Сложность создания** — требует времени и усилий  
❌ **Необходимость данных** — нужно собрать информацию из разных источников  
❌ **Субъективность** — может зависеть от предубеждений исследователя  
❌ **Ограниченная точность** — не всегда отражает реальность

## Процесс проведения

1. **Определение цели** — что хотим понять или улучшить
2. **Выбор сегмента** — определение целевой аудитории
3. **Сбор данных** — интервью, аналитика, поддержка
4. **Выделение этапов** — ключевые точки взаимодействия
5. **Создание карты** — визуализация пути клиента
6. **Анализ барьеров** — выявление проблемных зон
7. **Формулировка решений** — предложения по улучшению

## Компоненты CJM

| Компонент | Описание |
|-----------|----------|
| Персона | Типичный пользователь с его потребностями |
| Этапы пути | Ключевые точки взаимодействия |
| Действия | Что делает пользователь |
| Мысли/чувства | Эмоциональная кривая пользователя |
| Точки контакта | Каналы взаимодействия |
| Барьеры | Проблемы и боли |
| Возможности | Предложения по улучшению |

## Стоимость и сроки

- **Минимум**: 7 дней (для создания базовой карты)
- **Максимум**: 30+ дней (для глубокого исследования)
- **Доступ**: own/panel (требуется сбор данных)
- **Бюджет**: mid/high (зависит от глубины исследования)

## Сложность

**Medium-High** — требует сбора данных из разных источников и навыков визуализации.

---

*Метод относится к первому эшелону исследований.*
