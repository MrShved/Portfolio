# Product Designer Portfolio

Персональный сайт-портфолио для Product Designer.

## Технологии

- ⚛️ React
- 🎨 Tailwind CSS
- ⚡ Vite
- 🐙 Git

## Установка

```bash
npm install
```

## Запуск разработки

```bash
npm run dev
```

## Построение для продакшена

```bash
npm run build
```
